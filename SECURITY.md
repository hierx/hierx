# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `hierx`, please report
it by emailing alexander.hellervik@chalmers.se.

**Please do not open a public issue for security vulnerabilities.**

## Scope

This package is a scientific computing library. Security concerns are limited to:
- Arbitrary code execution via crafted inputs
- Denial of service via resource exhaustion
- Dependency vulnerabilities

## Response Timeline

- **Acknowledgement**: within 7 days
- **Assessment**: within 14 days
- **Fix release**: within 30 days for confirmed vulnerabilities
